http 简介

http 超文本传输协议，主要规定浏览器和服务器相互通信规则，它是基于TCP/IP通信协议来进行传递数据
浏览器发起一http请求， 首先建立TCP连接 也就是常说的三次握手， 接下来发起Http请求
一个http请求主要包括 请求行 请求头 请求正文
请求行：请求地址、请求的方法（get， post、put、delete）、请求状态
